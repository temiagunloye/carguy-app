## Roles & Responsibilities

### 1. Website Manager Agent
**Focus**: Design, UX, User Flow.
- **Weekly**: Audit density (keep it "Sorbet" calm).
- **Triggers**: New feature request -> Update `styles.css` consistency.
- **Output**: Updated `DESIGN_CONSISTENCY.md`.

### 2. Analytics Agent
**Focus**: Data integrity, GTM, GA4.
- **Weekly**: Verify GTM events firing (DebugView).
- **Triggers**: Drop in `cta_clicks` -> Audit `index.html` changes.
- **Output**: Updated `TRACKING_MAP.md`.

### 3. CRM / Growth Agent
**Focus**: Lead capture, Attribution, Automations.
- **Weekly**: Check "Detailed Analysis Log" in Dashboard.
- **Triggers**: Low TikTok conversion -> Check UTMs.
- **Output**: `AUTOMATION_TRIGGERS.md` and email templates.

## Operational Workflow
1.  **Monitor**: Review Dashboards (Monday).
2.  **Audit**: Check Consistency Checklist (Wednesday).
3.  **Execute**: Implement backlog items (Friday).
